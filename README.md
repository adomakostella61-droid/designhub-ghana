# DesignHub Ghana

🎨 **Creative Designs That Bring Your Ideas To Life**

A premium, modern, AI-powered graphic design marketplace built with Next.js, React, Node.js, and PostgreSQL.

## 🚀 Features

### Core Features
- **Design Services**: Logo, Flyers, Posters, Business Cards, Social Media Graphics, and more
- **Order Management**: Complete order lifecycle tracking from submission to delivery
- **Payment Integration**: Stripe, PayPal, MTN MoMo, Telecel Cash, AirtelTigo Money
- **AI Tools**: 8 AI-powered design assistance tools
- **Real-time Chat**: Customer-Designer communication
- **Portfolio Gallery**: Masonry gallery with lightbox preview
- **Reviews System**: Customer ratings and testimonials
- **Admin Dashboard**: Complete management system with analytics & price management
- **Blog System**: SEO-optimized design articles
- **WhatsApp Integration**: Floating widget for customer inquiries

### Admin Features
- ✅ **Dynamic Price Management** - Update all service prices in real-time
- ✅ **Orders Management**
- ✅ **Customer Management**
- ✅ **Revenue Analytics**
- ✅ **File Delivery System**
- ✅ **Message Management**

### Design Features
- Glassmorphism UI
- Dark/Light Mode
- Fully Responsive (Mobile First)
- Premium Animations (Framer Motion)
- SEO Optimized
- Fast Loading
